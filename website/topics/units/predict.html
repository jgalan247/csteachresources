Predict: Data Units & File Sizes

Make predictions about file sizes and unit conversions

How This Works

For each scenario, make a prediction BEFORE revealing the answer. This helps develop your intuition for data sizes and calculation skills.

1

Which is Larger?

Which of these represents MORE data?

  • Option A: 5,000 KB
  • Option B: 4 MB

Your prediction:

Answer: Option A (5,000 KB) is larger

Working:

  • 5,000 KB = 5,000 √∑ 1,000 = 5 MB
  • 4 MB stays as 4 MB
  • 5 MB > 4 MB, so Option A is larger by 1 MB

Image File Size

A digital photo has these properties:

  • Resolution: 800 √ó 600 pixels
  • Colour depth: 24 bits (true colour)

What is the approximate file size?

Your prediction:

Answer: About 1.4 MB

Calculation:

  • Total pixels = 800 √ó 600 = 480,000 pixels
  • Size in bits = 480,000 √ó 24 = 11,520,000 bits
  • Size in bytes = 11,520,000 √∑ 8 = 1,440,000 bytes
  • Size in MB = 1,440,000 √∑ 1,000,000 = 1.44 MB
3

Audio File Size

A music track has these properties:

  • Duration: 4 minutes
  • Sample rate: 44,100 Hz (CD quality)
  • Bit depth: 16 bits
  • Channels: Stereo (2 channels)

What is the approximate uncompressed file size?

Your prediction:

Answer: About 42 MB

Calculation:

  • Duration = 4 √ó 60 = 240 seconds
  • Size in bits = 44,100 √ó 240 √ó 16 = 169,344,000 bits
  • For stereo √ó 2 = 338,688,000 bits
  • Size in bytes = 338,688,000 √∑ 8 = 42,336,000 bytes
  • Size in MB = 42,336,000 √∑ 1,000,000 = 42.3 MB

This is why MP3 compression is useful - it reduces this to about 4 MB!

4

Text File Size

A novel has approximately:

  • 80,000 words
  • Average 5 characters per word (plus spaces)
  • ASCII encoding (8 bits per character)

What is the approximate file size as plain text?

Your prediction:

Answer: About 500 KB

Calculation:

  • Characters per word (including space) = 5 + 1 = 6
  • Total characters = 80,000 √ó 6 = 480,000 characters
  • Size in bits = 480,000 √ó 8 = 3,840,000 bits
  • Size in bytes = 3,840,000 √∑ 8 = 480,000 bytes
  • Size in KB = 480,000 √∑ 1,000 = 480 KB

Text is very efficient! That's why ebooks are so small compared to images or video.

5

Bits vs Bytes Trap

Your internet speed is advertised as 100 Mbps (megabits per second).

How long would it take to download a 100 MB file?

Your prediction:

Answer: 8 seconds

This is a common trap!

  • Internet speeds are in bits (Mbps)
  • File sizes are in bytes (MB)
  • 100 Mbps = 100 megabits per second
  • 100 MB = 100 megabytes = 800 megabits
  • Time = 800 Mb √∑ 100 Mbps = 8 seconds

Always check if it's bits (b) or bytes (B) - they differ by a factor of 8!

Storage Planning

A photographer wants to store 1,000 photos. Each photo is:

  • Resolution: 4000 √ó 3000 pixels
  • Colour depth: 24 bits

What size memory card do they need (minimum)?

Your prediction:

Answer: 32 GB (or 64 GB to be safe)

Calculation:

  • Pixels per photo = 4000 √ó 3000 = 12,000,000
  • Bits per photo = 12,000,000 √ó 24 = 288,000,000 bits
  • Bytes per photo = 288,000,000 √∑ 8 = 36,000,000 = 36 MB
  • Total for 1,000 photos = 36 √ó 1,000 = 36,000 MB = 36 GB

Since 36 GB > 32 GB, they need at least a 64 GB card (the next standard size).

Colour Depth Impact

Two identical images (1000 √ó 1000 pixels) are saved:

  • Image A: 8-bit colour depth (256 colours)
  • Image B: 24-bit colour depth (16.7 million colours)

How many times larger is Image B compared to Image A?

Your prediction:

Answer: 3 times larger

Calculation:

  • Image A = 1000 √ó 1000 √ó 8 = 8,000,000 bits = 1 MB
  • Image B = 1000 √ó 1000 √ó 24 = 24,000,000 bits = 3 MB
  • Ratio = 24 √∑ 8 = 3 times larger

The file size is directly proportional to colour depth - triple the bits means triple the size!

Real-World Estimation

A student types an essay with 2,000 words. Assuming:

  • Average 5 characters per word
  • Spaces between words
  • ASCII encoding (1 byte per character)

Your prediction:

Answer: About 80 essays

Calculation:

  • Characters per word (including space) = 6
  • Total characters per essay = 2,000 √ó 6 = 12,000
  • Bytes per essay = 12,000 bytes = 12 KB
  • Essays per MB = 1,000 KB √∑ 12 KB ‚âà 83 essays

Text is incredibly efficient - you could fit thousands of essays on a smartphone!

Developing intuition for file sizes takes practice. The more you work with these calculations, the better you'll become at estimating!
