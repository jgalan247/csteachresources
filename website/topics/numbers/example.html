Question

Example 6 Core

Binary Shifts

Question

a) Perform a left shift by 2 on 00010100 (20)

b) Perform a right shift by 2 on 01100100 (100)

Solution (a) - Left Shift

1 Move all bits left by 2 positions

Original: 0 0 0 1 0 1 0 0

After: 0 1 0 1 0 0 0 0

Fill empty positions on right with 0s

2 Calculate expected result

Left shift by 2 = multiply by 2¬≤ = multiply by 4

20 √ó 4 = 80

Answer (a)

00010100 << 2 = 01010000 (80)

Solution (b) - Right Shift

1 Move all bits right by 2 positions

Original: 0 1 1 0 0 1 0 0

After: 0 0 0 1 1 0 0 1

Fill empty positions on left with 0s, rightmost bits are lost

2 Calculate expected result

Right shift by 2 = divide by 2¬≤ = divide by 4

100 √∑ 4 = 25

Answer (b)

01100100 >> 2 = 00011001 (25)
