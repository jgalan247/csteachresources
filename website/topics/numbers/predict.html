Make predictions about number conversions before calculating

How This Works

For each question, make a prediction BEFORE revealing the answer. Building intuition for number systems helps you spot errors quickly.

1

Binary Range Check

The binary number 11111111 represents which denary value?

Your prediction:

Answer: 255

Working:

128 + 64 + 32 + 16 + 8 + 4 + 2 + 1 = 255

This is the maximum value that can be stored in 8 bits.

2

Binary to Denary

Convert the binary number 10101010 to denary.

Your prediction:

Answer: 170

Working:

128 + 0 + 32 + 0 + 8 + 0 + 2 + 0 = 170

The pattern 10101010 has 1s in the 128, 32, 8, and 2 positions.

3

Denary to Binary

Convert denary 200 to binary.

Your prediction:

Answer: 11001000

Working:

200 = 128 + 64 + 8 = 11001000

  • 128 fits (200-128=72), mark 1
  • 64 fits (72-64=8), mark 1
  • 32 doesn't fit, mark 0
  • 16 doesn't fit, mark 0
  • 8 fits (8-8=0), mark 1
  • Rest are 0s

Overflow Detection

Which of these additions will cause an overflow error in 8 bits?

Your prediction:

Answer: 200 + 100 causes overflow

Working:

  • 100 + 100 = 200 ‚úì (fits in 8 bits)
  • 127 + 127 = 254 ‚úì (fits in 8 bits)
  • 200 + 100 = 300 ‚úó (exceeds 255, overflow!)
  • 64 + 64 = 128 ‚úì (fits in 8 bits)

8 bits can only store values 0-255. Any result > 255 causes overflow.

Binary to Hexadecimal

Convert binary 11110000 to hexadecimal.

Your prediction:

Answer: F0

Working:

Split into nibbles: 1111 | 0000

1111 = 15 = F

0000 = 0 = 0

Result: F0

10

MSB and LSB

In the binary number 10110101, what is the value of the MSB (Most Significant Bit)?

Your prediction:

Answer: 1 (or it contributes 128 to the value)

Explanation:

The MSB is the leftmost bit. In 10110101, the MSB is 1.

The MSB has the highest place value. In 8-bit binary, if the MSB is 1, it contributes 128 to the total value.

The question asks for the VALUE of the MSB, which is 1.

Key Patterns to Remember

  • Powers of 2: 1, 2, 4, 8, 16, 32, 64, 128 - memorise these!
  • Max 8-bit value: 255 (11111111 in binary, FF in hex)
  • Hex digits A-F: A=10, B=11, C=12, D=13, E=14, F=15
  • Left shift: Multiplies by 2 per position
  • Right shift: Divides by 2 per position (integer division)
  • 4 bits = 1 hex digit: Split binary into groups of 4
