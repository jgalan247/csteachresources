Predict: Utility Software

Predict which utility software is needed and why

For each scenario, predict which utility software would be most appropriate and explain why. Think about what each utility does before revealing the answer.

Scenario 1

Protecting Confidential Documents

A solicitor needs to store sensitive client information on their laptop. They are concerned that if the laptop is lost or stolen, the confidential files could be read by whoever finds it.

Predict: Which utility software should be used, and why?

Answer: Encryption Software

Explanation: Encryption software should be used to scramble the confidential files so they cannot be read without the decryption key.

  • Even if the laptop is stolen, the data remains unreadable
  • Full disk encryption protects all files automatically
  • Helps meet legal requirements (GDPR) for protecting personal data
  • The solicitor keeps the decryption key secure, so only they can access the files
Scenario 2

Slow Computer Performance

A user notices their old desktop computer with a hard disk drive (HDD) is running slowly. Programs take a long time to load, and the computer makes frequent clicking noises when accessing files.

Predict: Which utility software could help, and why?

Answer: Defragmentation Software

Explanation: The clicking noises and slow loading suggest the read/write head is moving frequently to access fragmented files.

  • Files have become scattered across the disk over time
  • Defragmentation reorganises files into contiguous blocks
  • This reduces head movement and speeds up file access
  • Note: This only applies to HDDs, not SSDs
Scenario 3

Emailing Large Files

A photographer needs to send 50 high-resolution images (totalling 200MB) to a client via email. However, their email provider has a 25MB attachment limit.

Predict: Which utility software should be used, and why?

Answer: Data Compression Software

Explanation: Compression software can reduce the total file size to fit within the email limit.

  • ZIP compression can combine all 50 images into one archive
  • The compressed archive will be significantly smaller
  • May need to send in multiple compressed batches if still over 25MB
  • Lossy compression (JPEG) could further reduce image file sizes if quality loss is acceptable
Scenario 4

Running Out of Storage

A student has a 500GB hard drive that is nearly full. They have many old documents, photos, and videos that they want to keep but don't access frequently.

Predict: Which utility software could help free up space?

Answer: Data Compression Software

Explanation: Compressing infrequently accessed files saves storage space.

  • Old files can be compressed into ZIP archives
  • Documents often compress very well (50-80% reduction)
  • Videos and photos may have less compression potential if already compressed
  • Files can be decompressed when needed again
Scenario 5

New SSD Performance

A user has recently upgraded from an HDD to an SSD. Their IT friend suggests running the same utility they used on their old hard drive to "optimise performance".

Predict: Should they run defragmentation on their new SSD? Why or why not?

Answer: NO - Do NOT defragment an SSD

Explanation: Defragmentation is unnecessary and harmful for SSDs.

  • SSDs have no moving parts - they access any location equally fast
  • Defragmentation causes many unnecessary write operations
  • SSDs have limited write cycles - unnecessary writes reduce lifespan
  • Modern operating systems disable defrag for SSDs automatically
  • SSDs use TRIM instead to manage deleted data efficiently
Scenario 6

Secure Online Banking

A bank wants to ensure that when customers log into their online banking, their password and financial data cannot be intercepted by hackers monitoring the network.

Predict: Which utility technology protects this data?

Answer: Encryption

Explanation: Encryption protects data during transmission over networks.

  • HTTPS uses encryption to secure the connection
  • Data is encrypted before being sent across the internet
  • Even if intercepted, the data appears as meaningless characters
  • Only the bank's server can decrypt the information
  • Look for the padlock icon in browsers indicating encryption
Scenario 7

Slow File Saving

An office worker notices that saving large documents takes much longer than it used to on their desktop PC with a traditional hard drive. The computer has been in use for 3 years without any maintenance.

Predict: What utility might help, and what is the underlying problem?

Answer: Defragmentation Software

Explanation: Over 3 years, files have become fragmented, causing slow write operations.

  • Files and deletions over time create gaps in storage
  • New files get split across multiple locations
  • The HDD's read/write head must move to many positions
  • Defragmentation consolidates free space and files
  • Regular defragmentation (e.g., monthly) prevents this buildup
Scenario 8

Streaming Music Quality

A music streaming service wants to deliver songs quickly to users, even those with slow internet connections. They need to reduce the file size of audio tracks while maintaining acceptable listening quality.

Predict: Which type of compression would they use, and why?

Answer: Lossy Compression (e.g., MP3, AAC)

Explanation: Lossy compression dramatically reduces audio file sizes.

  • Removes audio frequencies humans can't easily hear
  • Can reduce files by 90% compared to uncompressed audio
  • Quality loss is usually acceptable for casual listening
  • Enables faster streaming with less bandwidth
  • Different quality settings let users choose file size vs quality balance
Scenario 9

USB Drive Security

A business executive travels frequently with a USB drive containing confidential business plans and financial forecasts. They are worried about what would happen if they lose the USB drive in an airport.

Predict: Which utility should be used to protect this data?

Answer: Encryption Software

Explanation: Encrypting the USB drive protects data if the drive is lost.

  • Many USB drives have built-in encryption features
  • Third-party encryption software can also be used
  • Data is unreadable without the password or key
  • BitLocker (Windows) or FileVault (Mac) can encrypt drives
  • Even if found, the contents remain confidential
Scenario 10

Source Code Backup

A software developer wants to compress their project folder containing 1000 text-based source code files for backup. They need to ensure they can restore the exact original files later.

Predict: Which type of compression should they use, and why?

Answer: Lossless Compression (e.g., ZIP)

Explanation: Source code must be preserved exactly - any data loss would break the programs.

  • Lossy compression removes data permanently
  • Even one changed character could cause code errors
  • Text files compress very well with lossless methods (often 70-90%)
  • ZIP preserves the exact original bytes
  • The compressed archive can be extracted to get identical files
Scenario 11

Housekeeping Tasks

A teacher explains that utility software performs "housekeeping tasks" on a computer. A student asks what this means and why the operating system doesn't do these tasks itself.

Predict: How would you explain this?

Answer: Utility software provides additional maintenance the OS doesn't include

Explanation:

  • Housekeeping tasks = maintenance and optimisation activities
  • The OS handles core functions: memory, processes, files, hardware
  • Utility software adds extra features for security and performance
  • Examples: encryption (security), defragmentation (optimisation), compression (efficiency)
  • Some utilities come pre-installed, others can be added later
  • Third-party utilities often have more features than built-in options
Scenario 12

Website Image Optimisation

A web developer has uploaded high-quality photographs to a website, but pages are loading slowly. They need to reduce image file sizes to improve loading speed while keeping images looking good.

Predict: Which compression approach would be best?

Answer: Lossy Compression (JPEG)

Explanation: JPEG lossy compression is ideal for photographs on websites.

  • Photographs don't need to be pixel-perfect on websites
  • JPEG compression can reduce images by 80-95%
  • Quality settings let developers balance size vs appearance
  • Smaller images = faster page loading = better user experience
  • For logos/graphics with sharp edges, PNG (lossless) may be better
