How This Works

For each scenario, predict the best answer before revealing the explanation. This helps you develop intuition about when and how to use different compression types.

1 Compression Type

Text Document Compression

You need to email a 50-page Word document to your teacher.

Which compression type should you use?

Answer: Lossless

Text documents require lossless compression (like ZIP). Every character and formatting detail must be preserved exactly. Losing even a single character could change the meaning!

2 Compression Type

Holiday Photos for Social Media

You want to share holiday photos on Instagram. The original photos are 25MB each.

Which compression type is most appropriate?

Answer: Lossy (JPEG)

Social media photos don't need perfect quality. JPEG (lossy) will dramatically reduce file size (to ~2-3MB) while looking virtually identical on screen. Instagram will compress them further anyway!

3 File Effects

MP3 Compression

You convert a WAV music file to MP3 format.

What happens to the file?

Answer: Smaller file, data permanently removed

MP3 is lossy compression. It removes audio frequencies that humans typically can't hear or don't notice. The removed data is gone forever - you can't convert an MP3 back to the original WAV quality.

4 Compression Type

Software Distribution

A company needs to distribute a software installer that's 500MB.

Which compression type must they use?

Answer: Lossless

Software must use lossless compression. A single bit changed in executable code could crash the program or create security vulnerabilities. ZIP is commonly used for software distribution.

5 File Effects

Re-saving a JPEG

You open a JPEG image, make no changes, and save it again as JPEG.

What happens to the image quality?

Answer: Quality gets worse

This is called generation loss. Each time you save a JPEG, lossy compression removes more data. After many re-saves, the image noticeably degrades. This is why photographers keep original files!

6 Scenario

Video Streaming

Netflix streams 4K movies to millions of users simultaneously.

What type of compression do they use?

Answer: Lossy

Video streaming requires lossy compression. An uncompressed 4K movie would be about 1TB! Codecs like H.264 and H.265 compress video to manageable sizes while maintaining acceptable quality.

7 File Effects

ZIP File Extraction

You compress a folder of documents into a ZIP file, then extract it.

What do you get back?

Answer: Exact original files

ZIP uses lossless compression. When you extract the files, you get back exactly what you put in - every bit is identical to the original. That's the key property of lossless compression!

8 Compression Type

Medical X-Ray Images

A hospital needs to store and transmit X-ray images for diagnosis.

Which compression type should they use?

Answer: Lossless

Medical images must use lossless compression. Doctors need to see every detail for accurate diagnosis. Lossy compression could hide subtle signs of disease or injury.

9 File Size

Compression Ratio

A 100MB photo is compressed to JPEG and also to PNG.

Which format produces the smaller file?

Answer: JPEG (lossy) is smaller

Lossy compression (JPEG) typically achieves much smaller file sizes than lossless (PNG) because it can discard data. A photo might be 3MB as JPEG but 30MB as PNG.

10 Compression Type

Logo for Website

A company logo has solid colours and sharp edges. It needs to go on a website.

Which format is best?

Answer: PNG

PNG is best for graphics with sharp edges and solid colours (logos, icons, diagrams). JPEG's lossy compression creates visible artifacts around sharp edges. PNG keeps them crisp!

11 Advantages

Main Advantage of Lossless

What is the PRIMARY advantage of lossless compression over lossy?

Answer: Original data perfectly preserved

The key advantage of lossless is that no information is lost. You can decompress and get back exactly what you started with. This is essential for text, code, and important documents.

12 Disadvantages

Main Disadvantage of Lossy

What is the PRIMARY disadvantage of lossy compression?

Answer: Quality loss is permanent and irreversible

Once lossy compression removes data, it's gone forever. You can't "decompress" a JPEG back to its original quality. Always keep original files if quality matters!
