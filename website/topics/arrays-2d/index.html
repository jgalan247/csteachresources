Key Vocabulary

2D array

An array of arrays - data organised in rows and columns, like a table or spreadsheet.

row

A horizontal line in a 2D array. The first index selects which row.

column

A vertical line in a 2D array. The second index selects which column.

nested loop

A loop inside another loop. Used to visit every element in a 2D array.

grid

Another name for a 2D array. Think of game boards, maps, or spreadsheets.

matrix

A mathematical term for a 2D array of numbers. Used in maths and graphics.

Quick Reference

arrays_2d_examples.py 
# Creating a 2D array (3 rows x 4 columns)
grid = [
    [1,  2,  3,  4],   # Row 0
    [5,  6,  7,  8],   # Row 1
    [9, 10, 11, 12]    # Row 2
]

# Accessing elements: array[row][column]
print(grid[0][0])  # First row, first column: 1
print(grid[1][2])  # Second row, third column: 7
print(grid[2][3])  # Third row, fourth column: 12

# Getting a whole row
print(grid[1])     # [5, 6, 7, 8]

# Number of rows and columns
rows = len(grid)           # 3 rows
cols = len(grid[0])        # 4 columns

# Nested loop - print every element
for row in grid:
    for item in row:
        print(item, end=" ")
    print()  # New line after each row

# Nested loop with indices
for r in range(len(grid)):
    for c in range(len(grid[r])):
        print(grid[r][c], end=" ")
