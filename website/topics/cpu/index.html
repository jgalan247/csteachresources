Key Vocabulary

CPU (Central Processing Unit)
The "brain" of the computer that processes all instructions
Control Unit (CU)
Coordinates and manages all CPU operations
ALU (Arithmetic Logic Unit)
Performs arithmetic and logical operations
Cache
Small, fast memory for frequently used data
Registers
Tiny, super-fast memory inside the CPU (PC, MAR, MDR, ACC)
RAM
Main memory where programs and data are stored
Fetch-Execute Cycle
The continuous process of getting and running instructions
Von Neumann Architecture
Design where programs and data share the same memory
Clock Speed
Number of cycles per second, measured in GHz
Core
Independent processing unit within the CPU
