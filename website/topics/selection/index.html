Key Vocabulary

selection

Choosing which code to run based on a condition. Also called branching.

condition

An expression that is either True or False. age >= 18

if statement

Runs code only if a condition is True. if score > 50:

else

Code that runs when the if condition is False.

elif

Short for "else if". Checks another condition if the first was False.

comparison operator

Symbols that compare values: == != < > <= >=

Boolean

A value that is either True or False. Named after mathematician George Boole.

and operator

Combines conditions: both must be True. x > 0 and x < 10

or operator

Combines conditions: at least one must be True. x == 0 or x == 1

not operator

Reverses a condition. not logged_in

Quick Reference

selection_preview.py 
# Simple if statement
age = 15
if age >= 13:
    print("You're a teenager!")

# if-else: two options
score = 45
if score >= 50:
    print("Pass")
else:
    print("Fail")

# if-elif-else: multiple options
grade = 75
if grade >= 90:
    print("A")
elif grade >= 70:
    print("B")
elif grade >= 50:
    print("C")
else:
    print("Fail")

# Using Boolean operators
age = 25
has_licence = True

# and: both conditions must be True
if age >= 18 and has_licence:
    print("You can drive")

# or: at least one must be True
if age < 12 or age > 65:
    print("Free ticket")

# not: reverses the condition
logged_in = False
if not logged_in:
    print("Please log in")
