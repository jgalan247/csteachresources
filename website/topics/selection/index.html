Key Vocabulary

selection

Choosing which code to run based on a condition. Also called branching.

condition

An expression that is either True or False. age >= 18

if statement

Runs code only if a condition is True. if score > 50:

else

Code that runs when the if condition is False.

elif

Short for "else if". Checks another condition if the first was False.

comparison operator

Symbols that compare values: == != < > <= >=

Quick Reference

selection_preview.py 
# Simple if statement
age = 15
if age >= 13:
    print("You're a teenager!")

# if-else: two options
score = 45
if score >= 50:
    print("Pass")
else:
    print("Fail")

# if-elif-else: multiple options
grade = 75
if grade >= 90:
    print("A")
elif grade >= 70:
    print("B")
elif grade >= 50:
    print("C")
else:
    print("Fail")
